\begin{block}{科学问题}
    果蝇左右脑蘑菇体是否只是镜像结构，还是存在可传播到记忆环路的递质侧化？本项目用 FlyWire 连接组约束的透明赛博果蝇代理仿真框架，将“右侧 5-HT 富集、左侧 Glu 偏置”的结构发现推进到 DPM 光遗传和 OCT/MCH 行为实验预测。
  \end{block}
  \refs{FlyWire: Dorkenwald et al., Nature 2024; Schlegel et al., Nature 2024. Mushroom body learning circuit: Aso et al., eLife 2014; Modi et al., Annu. Rev. Neurosci. 2020.}
\end{frame}

\begin{frame}{汇报结构与结论层级}
\small
\begin{columns}[T,onlytextwidth]
  \begin{column}{0.50\textwidth}
    \begin{block}{四个问题}
      \begin{enumerate}
        \item KC 输入是否存在稳定递质侧化？
        \item 侧化 seed 是否能传播到记忆相关 readout？
        \item DPM 光遗传能否给出可测 release pattern？
        \item OCT/MCH 行为任务是否能承接这些预测？
      \end{enumerate}
    \end{block}
  \end{column}
  \begin{column}{0.46\textwidth}
    \begin{block}{当前结论}
      \begin{itemize}
        \item \orange{结构发现强}：5-HT 右偏，Glu 左偏，$\alpha'\beta'$ 最强。
        \item \orange{传播结果支持功能相关性}：readout 到 DAN、APL、DPM、MBON/MBIN。
        \item \orange{行为代理可用但有限}：OCT/MCH 方向稳定，MB 扰动未显著。
      \end{itemize}
    \end{block}
  \end{column}
\end{columns}
\end{frame}

\begin{frame}{实验 1：结构层递质侧化统计}
\small
\begin{columns}[T,onlytextwidth]
  \begin{column}{0.48\textwidth}
    \begin{block}{数据与对象}
      \begin{itemize}
        \item 数据：FlyWire v783 neurons、synapses、cell annotations。
        \item 递质：per-synapse neurotransmitter prediction。
        \item 对象：左右半脑 Kenyon cells（KC），按 KC subtype 分组。
      \end{itemize}
    \end{block}
  \end{column}
  \begin{column}{0.48\textwidth}
    \begin{block}{分析和指标}
      \begin{itemize}
        \item 对每个 KC 计算上游输入递质比例。
        \item 左右半脑比较：Cohen's $d$，正值代表右偏，负值代表左偏。
        \item 统计控制：Mann--Whitney、permutation、bootstrap、FDR。
        \item 术语：5-HT 是血清素；Glu 是谷氨酸；$\alpha'\beta'$ 是记忆巩固相关 KC lobe。
      \end{itemize}
    \end{block}
  \end{column}
\end{columns}
\refs{Connectome and annotation basis: Dorkenwald et al., Nature 2024; Schlegel et al., Nature 2024. NT prediction: Eckstein et al., Cell 2024.}
\end{frame}

\begin{frame}{结果 1：KC 输入中 5-HT 右偏、Glu 左偏}
\scriptsize
\begin{columns}[T,onlytextwidth]
  \begin{column}{0.58\textwidth}
    \includegraphics[width=0.96\textwidth]{Fig_ext1_cohens_d_heatmap_all_NT.png}

  \begin{column}{0.38\textwidth}
    \begin{block}{主要观察}
      \begin{itemize}
        \item 5-HT/Serotonin：11/11 KC 类型右偏。
        \item Glu/Glutamate：7/11 KC 类型左偏且显著。
        \item 最强效应集中在 $\alpha'\beta'$。

      \end{itemize}
    \end{block}
    \begin{block}{结论}
      左右蘑菇体不是简单镜像；至少在 KC 输入的化学组成上存在稳定侧化。
    \end{block}
  \end{column}

\begin{frame}{实验 2：连接组传播算法与功能可达性（赛博果蝇仿真）}
\small
\begin{columns}[T,onlytextwidth]
  \begin{column}{0.51\textwidth}
    \begin{block}{图传播算法}
      \begin{itemize}
        \item 构图：神经元为节点，突触数/连接强度为边权，兴奋/抑制符号进入 signed weight。
        \item 初始化：把右 5-HT-rich KC 或左 Glu-rich KC 设为 seed vector $x_0$。
        \item 传播：$x_{t+1}=\hat{A}^{T}x_t$，最多 3 hop；$\hat{A}$ 是归一化连接矩阵。
        \item 汇总：对每个 hop 的响应求和，得到各下游脑区的 response mass。
        \item 对照：同侧、同数量随机 KC 重复传播，形成 null distribution。
      \end{itemize}
    \end{block}
  \end{column}
  \begin{column}{0.45\textwidth}
    \begin{block}{方法合理性}
      \begin{itemize}
        \item 这不是“环境里随机游走”，而是在真实连接组图上的加权信号传播。
        \item 逻辑类似网络扩散/图传播：先问结构差异能不能到达目标模块。
        \item 如果到达 DAN、APL、DPM、MBON/MBIN，才有资格讨论学习、记忆维持和输出读出。
        \item 结论边界：证明功能可达性，不等于证明真实神经放电或真实行为。
      \end{itemize}
    \end{block}
  \end{column}
\end{columns}
\refs{Weighted connectome graph analyses are standard in large-scale connectomics; see Scheffer et al., eLife 2020; Dorkenwald et al., Nature 2024. MB circuit readouts: Aso et al., eLife 2014; Li et al., eLife 2020.}
\end{frame}

\begin{frame}{实验 2：传播读出指标与统计检验（赛博果蝇仿真）}
\small
\begin{tabularx}{\textwidth}{@{}p{3.0cm}p{4.2cm}X@{}}
\toprule
\textbf{指标} & \textbf{计算含义} & \textbf{生物解释和用途} \\
\midrule
memory axis mass & 传播信号落到预定义记忆相关神经元集合的总量 & 类似图神经网络里某个子模块的 activation mass；越大表示侧化输入越容易影响记忆轴。\\
DAN 等 abs mass & 到达 DAN、APL、DPM、MBON 等神经元类别的信号强度，取绝对值，不区分正负 & DAN 像“教学信号”，APL/DPM 像“反馈和记忆维持”，MBON/MBIN 像“读出/调制接口”；用于判断影响到哪个功能模块。\\
response laterality & $(R-L)/(R+L)$，正值偏右，负值偏左 & 判断传播后的影响是否仍保留左右脑偏向，而不只是整体变强。\\
empirical $p$ & 和随机 seed 形成的 null distribution 比较 & 问题是“这个结果比随机挑 KC 更特殊吗”。\\
FDR $q$ & 多个指标一起检验后的假阳性校正 & 避免在很多脑区/指标里挑一个偶然显著的结果。\\
\bottomrule
\end{tabularx}
\vspace{0.4em}
\begin{block}{判定标准}
只有当侧化 seed 的读出超过同侧随机 KC 对照，并经过 FDR 校正后仍显著，才作为后续 DPM 光遗传或行为实验的优先候选。
\end{block}
\end{frame}

\begin{frame}{实验 3：MB readout 到 DN/motor bridge（赛博果蝇仿真）}
\small
\begin{columns}[T,onlytextwidth]
  \begin{column}{0.48\textwidth}
    \begin{block}{为什么要接到 DN}
      DAN、APL、DPM、MBON 还主要是脑内学习/记忆模块。若要解释行为，信号最终需要到达 descending neurons（DN，下行神经元）或其它运动出口。DN 可以理解成“大脑给身体发命令”的接口。
    \end{block}
  \end{column}
  \begin{column}{0.48\textwidth}
    \begin{block}{仿真流程}
      \begin{itemize}
        \item Seed：left/right/bilateral MBON、DAN、APL/DPM、memory axis。
        \item 仍在 FlyWire signed graph 上做多跳传播。
        \item 只统计落到 DN annotation 的响应。
        \item 输出：DN 数量、absolute mass、signed mass、laterality index。
        \item motor primitive 只是透明 heuristic，不是 Eon 私有 DN-to-body 权重。
      \end{itemize}
    \end{block}
  \end{column}
\end{columns}
\end{frame}

\begin{frame}{实验 4：DPM 光遗传仿真验证（赛博果蝇仿真）}
\small
\begin{columns}[T,onlytextwidth]
  \begin{column}{0.48\textwidth}
    \begin{block}{术语和实验含义}
      \begin{itemize}
        \item DPM：投射到蘑菇体的调制神经元，和嗅觉记忆稳定性相关。
        \item 光遗传：让特定神经元表达感光蛋白，再用光控制其活性。
        \item Opsin：感光蛋白；CsChrimson/ReaChR 对红光敏感。
        \item 5-HT/KC readout：预测 KC 区域接收到的血清素相关读出。
      \end{itemize}
    \end{block}
  \end{column}
  \begin{column}{0.48\textwidth}
    \begin{block}{仿真设置}
      \begin{itemize}
        \item Seed：left/right DPM。
        \item 扫描：opsin、波长、频率、脉宽、时长、光强。
        \item Pulse：单次光刺激宽度；train：连续刺激序列。
        \item 控制：180 度旋转后按脑侧注册，区分脑侧偏向和相机角度伪影。
        \item 湿实验对应：DPM-driver $>$ CsChrimson/ReaChR，加 5-HT sensor 或 GCaMP 成像。
      \end{itemize}
    \end{block}
  \end{column}
\end{columns}
\refs{DPM and memory: Yu et al., Cell 2005; Keene and Waddell, Nat. Rev. Neurosci. 2007. CsChrimson: Klapoetke et al., Nat. Methods 2014. ReaChR: Lin et al., Nat. Neurosci. 2013.}
\end{frame}

    \begin{block}{我们做了什么}
      在行为代理中分别模拟左/右 MB 静默、左右对称化、左右交换，再与 WT 对照比较。

    \begin{alertblock}{结果含义}

\begin{block}{为什么这个负结果重要}
它说明当前行为代理还不够敏感，或者 MB 侧化对 OCT/MCH 选择的影响需要更强的 DN/motor 闭环、剂量响应或真实湿实验验证。
\end{block}
\end{frame}

\begin{frame}{湿实验验证路径：目的、设计和风险}
\small
\begin{columns}[T,onlytextwidth]
  \begin{column}{0.32\textwidth}
    \begin{block}{目的}
      仿真只能生成可检验假说；湿实验要证明结构侧化是否真实存在、DPM 激活是否产生脑侧 readout 差异、群体行为是否按预测方向改变。
    \end{block}
  \end{column}
  \begin{column}{0.32\textwidth}
    \begin{block}{设计思路}
      结构：GRASP/split-GFP 验证侧化接触。功能：DPM-driver $>$ CsChrimson/ReaChR，KC/MB 表达 5-HT sensor 或 GCaMP，并做 180 度旋转控制。行为：OCT/MCH T-maze，CS+ 左右平衡，测试 delayed/conflict 条件。
    \end{block}
  \end{column}
  \begin{column}{0.32\textwidth}
    \begin{block}{主要 concern}
      成像角度可能造成伪侧化；NT prediction 不是直接化学测量；成像后的果蝇不能继续做行为；群体 T-maze 不能知道单只果蝇的侧化程度。因此需要结构、功能、行为三条证据相互支撑。
    \end{block}
  \end{column}
\end{columns}
\vspace{0.35em}
\begin{center}
\blue{结构、功能、行为三条证据合在一起，才构成强论证。}
\end{center}
\refs{GRASP: Feinberg et al., Neuron 2008; Gordon and Scott, Neuron 2009. Fly olfactory behavior: Tully and Quinn, J. Comp. Physiol. A 1985.}
\end{frame}

\begin{frame}{风险与下一步}
\small
\begin{columns}[T,onlytextwidth]
  \begin{column}{0.49\textwidth}
    \begin{block}{主要风险}
      \begin{itemize}
        \item 不是 Eon 私有闭环系统复刻。
        \item NT prediction 不是直接化学测量。
        \item 行为代理对 MB 扰动还不敏感。
        \item OCT/MCH sensory encoder 仍简化。
      \end{itemize}
    \end{block}
  \end{column}
  \begin{column}{0.47\textwidth}
    \begin{block}{下一步}
      \begin{itemize}
        \item 加强 MBON/DAN/APL/DPM 到 DN 的在线控制接口。
        \item 做 dose response、symmetrization、swap 和 random controls。
        \item 优先推进 DPM 红光成像和 conflict T-maze。
        \item 主文中区分结构发现、功能预测和待湿实验验证。
      \end{itemize}
    \end{block}
  \end{column}
\end{columns}
\end{frame}

\begin{frame}{核心结论}
\small
\begin{block}{发现}
果蝇蘑菇体 KC 输入存在稳定递质侧化：右侧更偏 5-HT，左侧更偏 Glu，最强在与记忆巩固相关的 $\alpha'\beta'$ 区。
\end{block}
\begin{block}{方法贡献}
FlyWire 连接组约束的透明赛博果蝇代理仿真框架，将该结构发现推进到连接组传播、DPM 光遗传预测和 OCT/MCH 行为实验设计。
\end{block}
\begin{alertblock}{边界}
当前还不能说连接组单独涌现完整行为，也不能说 MB 侧化已经被真实行为因果证明。下一步需要 DPM 成像、GRASP/split-GFP 和群体行为完成闭环。
\end{alertblock}

\end{frame}
